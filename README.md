# HEMMS - Hospital Equipment Maintenance Management System

## Description
HEMMS is a PL/SQL-based system that tracks and manages hospital equipment maintenance schedules. It includes:
- Equipment table
- Maintenance schedule
- Repair history
- Audit logs
- Functions and procedures to check maintenance and log repairs
- Interactive PL/SQL menu system
